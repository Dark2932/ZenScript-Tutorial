# ZenScript Tutorial

### 引言

本篇教程将详细介绍ZenScript语言的语法格式。

鉴于国内没有对于ZenScript语言进行详细讲解得教程，且已有的教程也需要有一定的编程基础才能够读懂，我编写了本教程，带领各位魔改作者们学习ZenScript，从入门到精通。

本教程会持续更新，并修改错误之处，如果有疑问请访问 仓库 提交意见。
